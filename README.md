## Usage

### Dependencies

- `cmake` (`>=3.17`)
- `sdl2`
- `sdl2_image`
- `sdl2_ttf`
- `jsoncpp`

### Build

```sh
git clone --recurse-submodules https://github.com/shvixxl/arkanoid-nine-hells.git
cd arkanoid-nine-hells

cmake .
make

./arkanoid-nine-hells
```
